Amazon Macieは、機械学習とパターンマッチングを使用して機密データを発見し、データセキュリティリスクを可視化し、これらのリスクに対して自動保護を可能にするデータセキュリティサービスです。組織のAmazon Simple Storage Service（Amazon S3）データ資産のセキュリティ姿勢を管理するために、MacieはS3の一般目的バケットのインベントリを提供し、バケットのセキュリティおよびアクセス制御を自動的に評価・監視します。「もしMacieがあなたのデータのセキュリティやプライバシーに潜在的な問題を検出した場合、たとえばバケットが一般公開されるようになると、Macieはレビューおよび修正が必要な結果を生成します。Macieはまた、センシティブなデータの発見と報告を自動化し、あなたの組織がAmazon S3に保存しているデータについての理解を深める手助けをします。」機密データを検出するには、Macieが提供する組み込みの基準や技術、あなたが定義したカスタム基準、またはその両方の組み合わせを使用できます。MacieがS3オブジェクト内に機密データを検出すると、Macieは検出した機密データについて通知する所見を生成します。マシーは、発見事項に加えて、あなたのAmazon S3データのセキュリティ状況や、機密データがデータ資産内のどこに存在する可能性があるかについての洞察を提供する統計情報や情報を提供します。これらの統計情報や情報は、特定のS3バケットやオブジェクトのより深い調査を実施するための判断を導くことができます。Amazon Macie コンソールまたは Amazon Macie API を使用して、調査結果、統計、およびその他の情報をレビューおよび分析できます。また、Amazon EventBridge および AWS Security Hub との Macie 統合を活用して、他のサービス、アプリケーション、システムを用いて調査結果を監視、処理、および改善することもできます。
